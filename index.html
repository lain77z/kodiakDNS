Usage:

$ perl kodiakDNS.pl domain

Features:

  • Domain query nameservers NS, MX, A
  • Query AXFR or bruteforce subdomains
  • Reverse DNS for IP Range
  • Domain DNS version
  • AS numbers
  • IPs in DNSBL
